OpenTWEMP is an Open-Source Project


OpenTWEMP is abbreviated as "Open Total War Engine Modding Platform".
This project develops an open-source implementation of classic Total War (M2TW/RTW) game engines and its related modding infrastructure.
The OpenTWEMP is created to port existing successful modification projects from legacy Creative Assembly game engine to a modern game platform.
As a founder of the project, I'm going to create a game engine which would be familiar and friendly both gamers and modmakers.

OpenTWEMP is a Modding Platform


The OpenTWEMP is planned to be combined from several software parts which will be create the one game engine platform:


Community Browser
 This is a desktop client to provide convenient and flexible mod management for existing Total War Engine mods.
Community Website
 This is a project to extend and support the official website for the OpenTWEMP project.
Game Editor
 This is a project of the united modding IDE including known existing GUI-based toolkits created by Total War community.
Game Engine
 This project will be implement a modern game engine inspired by classic Total War Engine from Creative Assembly.
Game SDK
 This is a project of the game SDK compatible with known modding tools to export/import game assets of classic Total War engines.

Currently, public repositories exist only for Community Browser and Community Website projects.

OpenTWEMP Project Roadmap


I plan to release the first beta version of the OpenTWEMP Community Browser in 2024.
Currently, I'm also investigating scope of both OpenTWEMP Game Editor and OpenTWEMP Game SDK projects.
I'm going to release the first alpha release Game Editor + SDK projects in 2025 year.
The OpenTWEMP Game Engine project is in planning phase, its first alpha release may be in 2025-2026.
