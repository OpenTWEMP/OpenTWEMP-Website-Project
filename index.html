What is OpenTWEMP ?

The "OpenTWEMP" is abbreviated as the "Open Total War Engine Modding Platform".
The OpenTWEMP project develops an open-source implementation of classic Total War game engine (M2TW/RTW) and its related modding infrastructure.
The OpenTWEMP is an attempt to create a modern game engine platform which would be well known to both gamers and modders of old TWE games.
The crucial mission of the OpenTWEMP project is porting existing game modifications created by the global Total War community.

OpenTWEMP as a Global Project
