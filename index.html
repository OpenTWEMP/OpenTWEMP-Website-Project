What is OpenTWEMP ?

The "OpenTWEMP" is abbreviated as the "Open Total War Engine Modding Platform".
The OpenTWEMP project develops an open-source implementation of classic Total War game engine (M2TW/RTW) and its related modding infrastructure.
The OpenTWEMP is an attempt to create a modern game engine platform which would be well known to both gamers and modders of old TWE games.
The crucial mission of the OpenTWEMP project is porting existing game modifications created by the global Total War community.

OpenTWEMP as a Global Project

The OpenTWEMP project is planned to be combined from several software parts which will be joined to the one game engine platform:

Community Browser
 This is a desktop client to provide convenient and flexible mod management for existing Total War Engine mods.
Community Website
 This is a project to extend and support the official website for the OpenTWEMP project.
Game Editor
 This is a project of the united modding IDE including known existing GUI-based toolkits created by Total War community.
Game Engine
 This project will be implement a modern game engine inspired by classic Total War Engine from Creative Assembly.
Game SDK
 This is a project of the game SDK compatible with known modding tools to export/import game assets of classic Total War engines.
